# 如何注册 GitHub 账号

本文来源：https://www.getaoning.com/archives/242在新窗口打开

打开 GitHub在新窗口打开官网，点击右上角的Sign up，进入注册页面。

依次输入自己的邮箱、密码、GitHub 用户名，并通过实人认证，创建 GitHub 账户。

从邮箱中接收验证码并填入 GitHub。

可以直接跳过个性化设置。

成功注册你的个人 GitHub 账户。

在 GitHub 上编辑此页 在新窗口打开
