# 贡献指南

正如你所看到，本站所有内容都是开源的。任何人都可以向本仓库贡献教程、文章。

若您希望为本站提供资源，请确保您的系统已安装node.js在新窗口打开及yarn在新窗口打开。

以下为本项目的相关命令：

yarn              # 安装依赖
yarn docs:dev     # 本地运行项目
yarn docs:build   # 将项目构建为静态 HTML 文件
